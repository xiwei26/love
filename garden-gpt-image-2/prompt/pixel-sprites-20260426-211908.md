# GPT Image 2 Prompt Pack: Pixel Sprite Regeneration From Original Photo-To-Pixel Direction

Use these prompts as separate `gpt-image-2` image generation jobs. Keep the existing filenames and replace only if you decide to generate assets later.

Source project record:

- `docs/superpowers/specs/2026-04-19-proposal-site-design.md` says the site uses custom 8-bit characters based on the couple.
- Female lead identifiers: long hair, purple coat, gentle posture.
- Male lead identifiers: glasses, dark outerwear, optional simplified backpack or camera hint.
- `docs/superpowers/plans/2026-04-19-proposal-site.md` repeats the production checklist: `hero-girl-*` must read as long-haired and wearing a purple coat; `hero-boy-*` must read as wearing glasses and dark outerwear.

Recommended workflow for the four couple sprites:

- Use image-to-image / image edit mode with the relevant source photo(s) of the couple as visual references.
- Preserve recognizability through silhouette, hair shape, glasses, coat colors, and posture rather than facial detail.
- Convert the person in the reference image into a classic Mario-era 8-bit character sprite, not a new invented character.
- If no reference image is available in the image tool, use the identifier notes in each prompt as the fallback.

Global output constraints for every sprite:

- Output as PNG with transparent background.
- Target asset size is exactly 128x128 pixels.
- If the image tool cannot output 128x128 directly, generate a square transparent PNG, then crop/resize to 128x128 using nearest-neighbor scaling.
- Single centered sprite only, no text, no logo, no frame, no ground shadow, no background scene.
- Crisp 8-bit / early-16-bit pixel art inspired by classic Mario-era sprites, hard-edged pixels, limited retro palette, readable silhouette at small size.
- Consistent pixel fairytale proposal style across the full set: deep blue/violet shadows, warm gold highlights, gentle romantic mood.
- Keep transparent padding around the sprite so it can animate inside the existing stage layout.
